Effets CSS Avancés

Introduction

CSS offre de nombreuses fonctionnalités pour créer des effets visuels avancés sans recourir à des images. Explorez les dégradés, les ombres et les transformations pour donner vie à vos designs web.

Dégradés CSS (Gradients)

Les dégradés permettent de créer des transitions douces entre deux couleurs ou plus.

Dégradé Linéaire

90°
Dégradé Linéaire
CSS

Le dégradé linéaire (linear-gradient) crée une transition progressive entre deux ou plusieurs couleurs le long d'une ligne droite.

Syntaxe:

background: linear-gradient(angle, color1, color2, ...);

L'angle détermine la direction du dégradé (0° = de bas en haut, 90° = de gauche à droite).

Dégradé Radial

Dégradé Radial
CSS

Le dégradé radial (radial-gradient) crée une transition entre deux ou plusieurs couleurs qui rayonnent à partir d'un point central.

Syntaxe:

background: radial-gradient(shape size at position, color1, color2, ...);

La forme peut être un cercle ou une ellipse, et la position définit le centre du dégradé.

Ombres CSS

Les ombres ajoutent de la profondeur et de la dimension à vos éléments.

Ombre de Boîte
CSS

La propriété box-shadow ajoute des ombres à un élément.

Syntaxe:

box-shadow: [inset] x-offset y-offset blur-radius spread-radius color;
  • inset: si présent, l'ombre sera à l'intérieur de la boîte
  • x-offset: décalage horizontal de l'ombre
  • y-offset: décalage vertical de l'ombre
  • blur-radius: rayon de flou
  • spread-radius: étalement de l'ombre
  • color: couleur de l'ombre

Ombre de Texte
CSS

La propriété text-shadow ajoute des ombres au texte.

Syntaxe:

text-shadow: x-offset y-offset blur-radius color;
  • x-offset: décalage horizontal de l'ombre
  • y-offset: décalage vertical de l'ombre
  • blur-radius: rayon de flou
  • color: couleur de l'ombre

Transformations CSS

Les transformations permettent de modifier la position, la taille et la forme des éléments.

Transformation
CSS

La propriété transform permet d'appliquer des transformations 2D ou 3D à un élément.

Fonctions courantes:

  • rotate(angle): fait pivoter un élément autour de son origine
  • scale(x, y): change la taille d'un élément
  • translate(x, y): déplace un élément
  • skew(x-angle, y-angle): incline un élément

Ces transformations peuvent être combinées: transform: rotate(45deg) scale(1.5) translate(10px, 20px);

Transform Origin

Origine
CSS

La propriété transform-origin définit le point d'origine des transformations.

Par défaut, l'origine est au centre de l'élément (50% 50%).

Modifier cette origine change le comportement des transformations, notamment pour les rotations et les mises à l'échelle.
