Le Modèle de Boîte CSS

Le modèle de boîte est l'un des concepts fondamentaux du CSS. Il décrit comment chaque élément HTML est représenté comme une boîte rectangulaire, avec des propriétés qui déterminent sa taille, sa position et son apparence.

Cette démonstration interactive vous permet de comprendre comment fonctionne le modèle de boîte et comment les propriétés CSS affectent les dimensions et l'apparence des éléments.

Éléments du Modèle de Boîte

Le modèle de boîte CSS comprend:

  • Content (Contenu) - Le contenu réel de l'élément (texte, images, etc.)
  • Padding (Rembourrage) - L'espace entre le contenu et la bordure
  • Border (Bordure) - La ligne qui entoure le contenu et le padding
  • Margin (Marge) - L'espace transparent autour de la bordure
.element {
  /* Propriétés du contenu */
  width: 100px;
  height: 100px;

  /* Propriétés de padding */
  padding: 20px;
  /* ou padding-top, padding-right, padding-bottom, padding-left */

  /* Propriétés de bordure */
  border: 5px solid black;
  /* ou border-width, border-style, border-color */

  /* Propriétés de marge */
  margin: 20px;
  /* ou margin-top, margin-right, margin-bottom, margin-left */
}

Box-sizing: Content-box vs Border-box

La propriété box-sizing détermine comment la largeur et la hauteur d'un élément sont calculées:

Content-box (comportement par défaut)

Les propriétés width et height s'appliquent uniquement au contenu de l'élément.

Largeur totale = width + padding + border

.box {
  width: 100px;
  padding: 20px;
  border: 5px solid black;
  box-sizing: content-box; /* par défaut */
}
/* Largeur totale = 100 + 40 + 10 = 150px */

Border-box

Les propriétés width et height incluent le padding et la bordure.

Largeur totale = width

.box {
  width: 100px;
  padding: 20px;
  border: 5px solid black;
  box-sizing: border-box;
}
/* Largeur totale = 100px */
