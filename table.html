Comment lire ce tableau ?

Chaque élément HTML est identifié par une étiquette colorée :

  • <table> - L'élément principal qui contient tout le tableau
  • <thead> - Contient les en-têtes du tableau
  • <tbody> - Contient les données du tableau
  • <tr> - Représente une ligne du tableau
  • <th> - Représente une cellule d'en-tête
  • <td> - Représente une cellule de données

Les étiquettes sont positionnées près de l'élément qu'elles identifient pour montrer clairement la structure.

Structure de base d'un tableau HTML

<table>
    <thead>
        <tr>
            <th>En-tête 1</th>
            <th>En-tête 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Donnée 1</td>
            <td>Donnée 2</td>
        </tr>
    </tbody>
</table>
