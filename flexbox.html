Guide Complet et Interactif de Flexbox CSS

Introduction

Flexbox (Flexible Box Layout) est un modèle de mise en page unidimensionnel qui permet de créer des designs flexibles et efficaces. Il simplifie considérablement la disposition, l'alignement et la distribution de l'espace entre les éléments.

Ce guide interactif vous permettra de comprendre et maîtriser toutes les facettes de Flexbox en manipulant directement les propriétés et en visualisant les résultats en temps réel.

1. Concepts fondamentaux

Avant de plonger dans les propriétés spécifiques, il est essentiel de comprendre les concepts de base qui sous-tendent Flexbox.

Terminologie de base

  • Conteneur flex (Flex container) : L'élément parent auquel on applique display: flex
  • Éléments flex (Flex items) : Les enfants directs du conteneur flex
  • Axe principal (Main axis) : L'axe sur lequel les éléments flex sont disposés principalement
  • Axe secondaire (Cross axis) : L'axe perpendiculaire à l'axe principal

Axe principal
Axe secondaire

Par défaut, l'axe principal va de gauche à droite (row), et l'axe secondaire de haut en bas. Ces directions peuvent être modifiées avec la propriété flex-direction.

Comment activer Flexbox

Pour convertir un élément en conteneur flex, on utilise la propriété display avec la valeur flex ou inline-flex.

/* Pour un conteneur de niveau bloc */ .container { display: flex; } /* Pour un conteneur de niveau en ligne */ .container { display: inline-flex; }

display: flex transforme l'élément en un bloc flex.

display: inline-flex transforme l'élément en un élément en ligne flex (comme inline-block mais avec des propriétés flex).

2. Propriétés du conteneur flex

Les propriétés suivantes s'appliquent au conteneur flex et déterminent comment ses enfants (les éléments flex) seront disposés.

flex-direction

Cette propriété établit l'axe principal, définissant ainsi la direction dans laquelle les éléments flex sont placés.

1
2
3
4
.container { display: flex; flex-direction: row; /* valeur par défaut */ }

row (défaut) : Les éléments sont disposés de gauche à droite.

row-reverse : Les éléments sont disposés de droite à gauche.

column : Les éléments sont disposés de haut en bas.

column-reverse : Les éléments sont disposés de bas en haut.

flex-wrap

Cette propriété détermine si les éléments flex doivent être affichés sur une seule ligne ou peuvent s'enrouler sur plusieurs lignes.

1

.container { display: flex; flex-wrap: nowrap; /* valeur par défaut */ }

nowrap (défaut) : Tous les éléments sont sur une seule ligne, même s'ils débordent.

wrap : Les éléments s'enroulent sur plusieurs lignes si nécessaire.

wrap-reverse : Les éléments s'enroulent sur plusieurs lignes dans le sens inverse.

flex-flow

Cette propriété est un raccourci pour flex-direction et flex-wrap.

1

.container { display: flex; flex-flow: row nowrap; /* valeur par défaut */ }

Le premier paramètre est la flex-direction, le second est le flex-wrap.

Cette propriété raccourcie simplifie l'écriture lorsque vous avez besoin de définir ces deux propriétés à la fois.

justify-content

Cette propriété aligne les éléments flex le long de l'axe principal.

1
2
3
.container { display: flex; justify-content: flex-start; /* valeur par défaut */ }

flex-start (défaut) : Les éléments sont alignés au début de l'axe principal.

flex-end : Les éléments sont alignés à la fin de l'axe principal.

center : Les éléments sont centrés sur l'axe principal.

space-between : Les éléments sont répartis uniformément; le premier élément est aligné au début, le dernier à la fin.

space-around : Les éléments sont répartis uniformément avec un espace égal autour de chacun.

space-evenly : Les éléments sont répartis uniformément avec un espace égal entre eux et les bords.

align-items

Cette propriété aligne les éléments flex le long de l'axe secondaire.

1
2
3
.container { display: flex; align-items: stretch; /* valeur par défaut */ }

stretch (défaut) : Les éléments sont étirés pour remplir le conteneur (respectant min/max-width/height).

flex-start : Les éléments sont alignés au début de l'axe secondaire.

flex-end : Les éléments sont alignés à la fin de l'axe secondaire.

center : Les éléments sont centrés sur l'axe secondaire.

baseline : Les éléments sont alignés sur leur ligne de base textuelle.

align-content

Cette propriété aligne les lignes d'éléments flex dans le conteneur quand il y a de l'espace supplémentaire sur l'axe secondaire (fonctionne uniquement s'il y a plusieurs lignes).

1
2
3

9
.container { display: flex; flex-wrap: wrap; align-content: stretch; /* valeur par défaut */ }

stretch (défaut) : Les lignes s'étirent pour occuper l'espace disponible.

flex-start : Les lignes sont rassemblées au début du conteneur.

flex-end : Les lignes sont rassemblées à la fin du conteneur.

center : Les lignes sont centrées dans le conteneur.

space-between : Les lignes sont réparties uniformément; la première ligne est au début, la dernière à la fin.

space-around : Les lignes sont réparties uniformément avec un espace égal autour de chacune.

Note: Cette propriété n'a d'effet que lorsque les éléments s'enroulent sur plusieurs lignes (flex-wrap: wrap).

3. Propriétés des éléments flex

Ces propriétés s'appliquent aux éléments enfants directs du conteneur flex (les flex items) et déterminent comment chaque élément se comporte individuellement.

order

Cette propriété contrôle l'ordre dans lequel les éléments apparaissent dans le conteneur flex.

1 (ordre: 3)
2 (ordre: 1)
3 (ordre: 2)
1
2
3
.item { order: 0; /* valeur par défaut */ }

La propriété order définit l'ordre d'apparition des éléments flex.

Par défaut, tous les éléments ont une valeur d'ordre de 0, et ils apparaissent dans l'ordre du code source.

Les valeurs plus basses apparaissent en premier. Les éléments avec la même valeur d'ordre apparaissent selon leur ordre dans le code source.

Les valeurs négatives sont autorisées et apparaîtront avant les éléments avec une valeur d'ordre plus élevée ou positive.

flex-grow

Cette propriété définit la capacité d'un élément à grandir si nécessaire. Elle détermine quelle proportion d'espace disponible dans le conteneur l'élément doit occuper.

Item 3:
1
2
3
.item { flex-grow: 0; /* valeur par défaut */ }

La propriété flex-grow accepte une valeur sans unité qui sert de proportion.

Elle détermine la quantité d'espace disponible à l'intérieur du conteneur flex que l'élément doit occuper.

Si tous les éléments ont flex-grow: 1, l'espace restant dans le conteneur sera réparti équitablement entre tous les enfants.

Si un élément a flex-grow: 2, il prendra deux fois plus d'espace que les éléments avec flex-grow: 1.

Une valeur de 0 (par défaut) signifie que l'élément ne grandira pas au-delà de sa taille de base.

flex-shrink

Cette propriété définit la capacité d'un élément à rétrécir si nécessaire.

Redimensionnez la fenêtre pour voir les effets de flex-shrink:

1

3
.item { flex-shrink: 1; /* valeur par défaut */ }

La propriété flex-shrink accepte une valeur sans unité qui sert de proportion.

Elle détermine dans quelle mesure l'élément rétrécira par rapport aux autres éléments flex quand il n'y a pas assez d'espace.

Si tous les éléments ont flex-shrink: 1, ils rétrécissent uniformément.

Si un élément a flex-shrink: 2, il rétrécira deux fois plus que les éléments avec flex-shrink: 1.

Une valeur de 0 signifie que l'élément ne rétrécira pas.

flex-basis

Cette propriété définit la taille initiale d'un élément flex avant que l'espace supplémentaire soit distribué.

flex-basis: auto
flex-basis: 150px
.item { flex-basis: auto; /* valeur par défaut */ }

La propriété flex-basis définit la taille initiale d'un élément flex. Elle peut être une longueur (px, em, etc.) ou auto.

auto : La taille est basée sur le contenu de l'élément ou sur sa largeur/hauteur définie.

0 : L'élément n'a pas de taille initiale, il se base uniquement sur flex-grow/shrink.

Unités de longueur (px, em, %) : Définit une taille spécifique comme point de départ.

Elle agit comme width quand flex-direction est row, et comme height quand flex-direction est column.

flex

Cette propriété est un raccourci pour flex-grow, flex-shrink et flex-basis combinés.

1
2
3
.item { flex: 0 1 auto; /* valeur par défaut */ }

La propriété flex est un raccourci qui combine trois propriétés: flex-grow, flex-shrink et flex-basis.

Syntaxe: flex: [flex-grow] [flex-shrink] [flex-basis]

flex: 1 = flex: 1 1 0 - L'élément peut grandir, rétrécir et n'a pas de taille de base.

flex: auto = flex: 1 1 auto - L'élément peut grandir, rétrécir et sa taille de base est auto.

flex: none = flex: 0 0 auto - L'élément ne peut ni grandir ni rétrécir.

flex: initial = flex: 0 1 auto - Valeur par défaut: l'élément ne peut pas grandir mais peut rétrécir si nécessaire.

align-self

Cette propriété permet de remplacer l'alignement par défaut (ou celui spécifié par align-items) pour des éléments flex individuels.

1
2
3
.container { display: flex; align-items: center; /* Alignement par défaut pour tous les éléments */ } .item { align-self: auto; /* valeur par défaut (hérite de align-items) */ }

La propriété align-self permet de remplacer l'alignement défini par align-items pour un élément spécifique.

Elle accepte les mêmes valeurs que align-items:

auto : Hérite de la valeur de align-items du parent (défaut).

flex-start : L'élément est aligné au début de l'axe secondaire.

flex-end : L'élément est aligné à la fin de l'axe secondaire.

center : L'élément est centré sur l'axe secondaire.

baseline : L'élément est aligné sur sa ligne de base textuelle.

stretch : L'élément est étiré pour remplir le conteneur (respectant min/max-width/height).
